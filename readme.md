# Chrome Extension: Expedia Reservation Checker

A Chrome extension to help you quickly reconcile Expedia reservations with Choice Advantage. It filters out canceled or no-show reservations and highlights mismatches, saving time for reconcilers and travel managers.

---

## Features

-- Load your Expedia reservation CSV and Choice Advantage data.  
-- Automatically check each reservation for discrepancies.  
-- Filter out canceled or no-show bookings.  
-- Quickly identify mismatched reservations to streamline reconciliation.  
-- Lightweight, fast, and easy-to-use interface.

---

## Installation

1. Clone or download this repository.  
2. Open Chrome and go to `chrome://extensions/`.  
3. Enable **Developer mode** (top-right toggle).  
4. Click **Load unpacked** and select the extension folder.  
5. The extension icon should appear in the toolbar.

---

## Usage

1. Open the extension.  
2. Upload your Expedia reservation CSV file.  
3. Upload your Choice Advantage CSV file.  
4. The extension will process the files and highlight any mismatches.  
5. Use the filters to ignore canceled or no-show reservations.
