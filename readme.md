# Chrome Extension: Booking.com Reservation Checker

A Chrome extension to help you quickly reconcile **Booking.com reservations** with Choice Advantage. It filters out canceled or no-show reservations and highlights mismatches, saving time for reconcilers and hotel management teams.

---

## Features

-- Load your Booking.com reservation XLS file and Choice Advantage data.  
-- Automatically check each reservation for discrepancies.  
-- Filter out canceled or no-show bookings.  
-- Quickly identify mismatched reservations to streamline reconciliation.  
-- Hover over any reservation to see all important details—including check-in/check-out changes, status updates, and group reservations consolidated under one guest.  
-- Lightweight, fast, and easy-to-use interface.  

---

## Installation

1. Clone or download this repository.  
2. Open Chrome and go to `chrome://extensions/`.  
3. Enable **Developer mode** (top-right toggle).  
4. Click **Load unpacked** and select the extension folder.  
5. The extension icon should appear in the toolbar.  

---

## Usage

1. Open the extension.  
2. Upload your Booking.com reservation XLS file.  
3. Upload your Choice Advantage data file.  
4. Click **Run Bot** — the extension will process the files and highlight any discrepancies.  
5. Use the filters to ignore canceled or no-show reservations, or scroll through all entries.  
6. Hover over any reservation line to see full details, including group reservations under a single guest.
